net-toolsのインストール - 非推奨だがとりあえずifconfig、arp、route、netstatを使いたい場合

ifconfigはLinuxでは、現在では非推奨のネットワーク情報の管理ツールになっていますが、とりあえずifconfigを使いたい場合は、net-toolsをインストールしましょう。

ifconfigは商用Unix環境と共通で使えるので、迷いたくないという場合も、とりあえずは、良いかもしれません。aptコマンドでインストールします。 

# net-toolsのインストール
sudo apt install -y net-tools

ifconfigコマンドがインストールされたことの確認

ifconfigコマンドがインストールされたことを確認しましょう。 

ifconfig

net-toolsに含まれているarp、route、netstatコマンドも使える用になります。

Linuxにおける正式なネットーワーク管理ツールはiproute2

現在Linuxにおける正式なネットーワーク管理ツールはiproute2となっています。(2020年6月8日記載)

iproute2には、ip・ss・bridge・rtacct・rtmon・tc・ctstat・lnstat・nstat・routef・routel・rtstat・tipc・arpd・devlinkというコマンドラインユーティリティが含まれています。

ifconfigで、IPアドレスをチェックしていた機能は「ip addr」コマンドを使います。 

ip addr
