curlのインストール

curlのインストール方法を解説します。aptコマンドを使って、curlをインストールします。sudoを使って管理者権限で実行します。 

sudo apt install -y curl

curlの主な用途

curlは、取得内容をデフォルトで標準出力に出力するHTTPクライアントです。デフォルトでインストールされているHTTPクライアントはwgetのことが多いですが、curlも手順で利用されることが多いので、インストールしておきます。
