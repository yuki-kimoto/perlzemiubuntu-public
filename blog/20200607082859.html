gitのインストール - Git分散型リポジトリ管理システム

分散型のリポジトリ管理システムであるGitを使える用にするにはgitパッケージをインストールします。aptコマンドで、sudoで管理者権限で実行します。 

# gitのインストール
sudo apt install -y git

gitの使い方についてはGit入門を参考にしてください。

gitコマンドがインストールされたことの確認

gitコマンドがインストールされたことを確認しましょう。 

git --help
