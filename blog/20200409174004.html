libpq-devのインストール - PostgreSQLクライアントのヘッダとライブラリ

PostgreSQLクライアントのヘッダやライブラリを含む「libpq-dev」をaptコマンドでインストールしましょう。

libpq-devは、PerlでDBIからPostgreSQLサーバーへ接続するためのDBD::Pgをインストールするために必要です。 

sudo apt install -y libpq-dev

インストールされた、PostgreSQLクライアントのヘッダファイルとライブラリの場所については、ヘッダーファイルの読み込みパスとライブラリの読み込みパスを参考にしてください。
